# 🏥 MEDISECURE

**Secure Authentication and Authorization Framework for Healthcare IoT Systems using IoT, Biometrics, and AI.**

## 📌 Overview

MEDISECURE is a healthcare IoT security solution designed to provide secure access to patient records using biometric authentication and AI-powered verification. The system enhances patient privacy, prevents unauthorized access, and streamlines healthcare workflows.

## ✨ Features

- 🔐 Fingerprint Authentication
- 👤 Face Recognition
- 🩺 Patient & Doctor Dashboard
- 📊 Secure Patient Record Management
- 🤖 AI-assisted Authentication
- 🌐 Web-based Interface
- 📡 IoT Device Integration
- 🔒 Role-Based Access Control

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python, Flask
- **Database:** MySQL
- **Hardware:** ESP32, Fingerprint Sensor, ESP32-CAM
- **Tools:** Arduino IDE, VS Code, GitHub

## 🚀 Installation

1. Clone the repository
```bash
git clone https://github.com/shafiyanaheed-maker/MEDISECURE.git
```

2. Navigate to the project folder
```bash
cd MEDISECURE
```

3. Install dependencies
```bash
pip install -r requirements.txt
```

4. Run the application
```bash
python app.py
```
